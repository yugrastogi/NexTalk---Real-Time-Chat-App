NexTalk is a Full-Stack, real-time web chat application.
I made it as a part of learning Full-Stack development, and getting an insight on how a chat application like "Whatsapp" works in real-time.
I learned a lot of thing making this project, like debugging, client-server architecture and how to implement a real time functionality using socket.io, etc.

Key Learnings
1.   Full-Stack Data Flow: Managing requests from the UI, processing them in the backend, and interacting with the database.
2.  Real-Time Communication: Implementing live messaging and client-side rendering using Socket.io.
3.  State Management: Handling complex global states seamlessly with Zustand.
4.  Security & Authentication: Securing user data and managing sessions using JWT and Bcryptjs.
